Forecasting Analysis Using LSTM and ARIMA

A project that uses ARIMA and LSTM models to predict future outcomes based on historical data.

This project focuses on forecasting analysis using LSTM and ARIMA models. The goal is to predict future weather parameters such as temperature, humidity, and pressure by analyzing recorded data from a weather monitoring system. The system is connected to the internet via IoT and the data is collected by a microcontroller ESP8266 and sent to ThingSpeak. The collected data is then analyzed and compared to meteorological data sheets to ensure accuracy. The end result is an online, wireless weather forecasting analysis that eliminates the need for manual monitoring.
